1. LINKED LIST BASICS

---

A Linked List is a linear data structure where elements are connected using pointers instead of contiguous memory.

Each node contains:

data
pointer to next node

Structure:

struct ListNode{
    int val;
    ListNode* next;
};

Example:

1 → 2 → 3 → 4 → 5 → NULL

Unlike arrays:

• Memory is not contiguous
• Nodes are connected through pointers
• Dynamic size

---

COMMON OPERATIONS

Traversal
Insertion
Deletion
Pointer manipulation

Traversal Example:

ListNode* temp = head;

while(temp != NULL){
    temp = temp->next;
}

---

WHEN LINKED LIST IS USED

• Dynamic memory allocation
• Frequent insert/delete operations
• Implement stacks and queues
• Graph adjacency list representation

2. REVERSE LINKED LIST

---

Problem:
Reverse a singly linked list.

Example:

1 → 2 → 3 → 4 → 5

Output:

5 → 4 → 3 → 2 → 1

---

PATTERN

Pointer Manipulation

We reverse the direction of next pointers.

---

APPROACH

Use three pointers:

prev
curr
next

Steps:

1. Store next node
2. Reverse pointer
3. Move pointers forward

---

ALGORITHM

prev = NULL
curr = head

while(curr != NULL)

    next = curr->next
    curr->next = prev

    prev = curr
    curr = next

Return prev

---

C++ CODE

ListNode* reverseList(ListNode* head){

    ListNode* prev = NULL;
    ListNode* curr = head;

    while(curr != NULL){

        ListNode* next = curr->next;

        curr->next = prev;

        prev = curr;
        curr = next;
    }

    return prev;
}

Time Complexity: O(n)
Space Complexity: O(1)

3. MIDDLE OF LINKED LIST

---

Problem:
Find the middle node of a linked list.

Example:

1 → 2 → 3 → 4 → 5

Output:

3

Even length case:

1 → 2 → 3 → 4 → 5 → 6

Output:

4 (second middle)

---

PATTERN

Fast and Slow Pointer

---

APPROACH

Use two pointers:

slow pointer → moves 1 step
fast pointer → moves 2 steps

When fast reaches the end,
slow will be at the middle.

---

C++ CODE

ListNode* middleNode(ListNode* head){

    ListNode* slow = head;
    ListNode* fast = head;

    while(fast != NULL && fast->next != NULL){

        slow = slow->next;
        fast = fast->next->next;
    }

    return slow;
}

Time Complexity: O(n)
Space Complexity: O(1)
