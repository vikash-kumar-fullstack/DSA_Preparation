1. LINKED LIST BASICS

---

A Linked List is a linear data structure where elements are connected using pointers instead of contiguous memory.

Each node contains:

data
pointer to next node

Structure:

struct ListNode{
    int val;
    ListNode* next;
};

Example:

1 → 2 → 3 → 4 → 5 → NULL

Unlike arrays:

• Memory is not contiguous
• Nodes are connected through pointers
• Dynamic size

---

COMMON OPERATIONS

Traversal
Insertion
Deletion
Pointer manipulation

Traversal Example:

ListNode* temp = head;

while(temp != NULL){
    temp = temp->next;
}

---

WHEN LINKED LIST IS USED

• Dynamic memory allocation
• Frequent insert/delete operations
• Implement stacks and queues
• Graph adjacency list representation

2. REVERSE LINKED LIST

---

Problem:
Reverse a singly linked list.

Example:

1 → 2 → 3 → 4 → 5

Output:

5 → 4 → 3 → 2 → 1

---

PATTERN

Pointer Manipulation

We reverse the direction of next pointers.

---

APPROACH

Use three pointers:

prev
curr
next

Steps:

1. Store next node
2. Reverse pointer
3. Move pointers forward

---

ALGORITHM

prev = NULL
curr = head

while(curr != NULL)

    next = curr->next
    curr->next = prev

    prev = curr
    curr = next

Return prev

---

C++ CODE

ListNode* reverseList(ListNode* head){

    ListNode* prev = NULL;
    ListNode* curr = head;

    while(curr != NULL){

        ListNode* next = curr->next;

        curr->next = prev;

        prev = curr;
        curr = next;
    }

    return prev;
}

Time Complexity: O(n)
Space Complexity: O(1)

3. MIDDLE OF LINKED LIST

---

Problem:
Find the middle node of a linked list.

Example:

1 → 2 → 3 → 4 → 5

Output:

3

Even length case:

1 → 2 → 3 → 4 → 5 → 6

Output:

4 (second middle)

---

PATTERN

Fast and Slow Pointer

---

APPROACH

Use two pointers:

slow pointer → moves 1 step
fast pointer → moves 2 steps

When fast reaches the end,
slow will be at the middle.

---

C++ CODE

ListNode* middleNode(ListNode* head){

    ListNode* slow = head;
    ListNode* fast = head;

    while(fast != NULL && fast->next != NULL){

        slow = slow->next;
        fast = fast->next->next;
    }

    return slow;
}

Time Complexity: O(n)
Space Complexity: O(1)

4. DETECT CYCLE IN LINKED LIST

---

Problem:
Check whether a linked list contains a cycle.

Example:

1 → 2 → 3 → 4 → 5
      ↑       ↓
      ← ← ← ←

Output:

true

---

PATTERN

Floyd's Cycle Detection Algorithm
(Fast-Slow Pointer)

---

APPROACH

Use two pointers:

slow pointer → moves 1 step
fast pointer → moves 2 steps

If there is a cycle,
fast will eventually meet slow.

If fast reaches NULL,
then no cycle exists.

---

C++ CODE

bool hasCycle(ListNode *head){

    ListNode* slow = head;
    ListNode* fast = head;

    while(fast != NULL && fast->next != NULL){

        slow = slow->next;
        fast = fast->next->next;

        if(slow == fast)
            return true;
    }

    return false;
}

Time Complexity: O(n)
Space Complexity: O(1)

5. MERGE TWO SORTED LINKED LISTS

---

Problem:
Merge two sorted linked lists into one sorted list.

Example:

List1
1 → 2 → 4

List2
1 → 3 → 4

Output

1 → 1 → 2 → 3 → 4 → 4

---

PATTERN

Two Pointer Merge

(Same idea as merge step in merge sort)

---

APPROACH

Maintain three pointers:

list1
list2
tail (merged list)

Use a dummy node to simplify head handling.

Steps:

1. Compare nodes of both lists
2. Attach the smaller node
3. Move pointer forward
4. Attach remaining nodes

---

C++ CODE

ListNode* mergeTwoLists(ListNode* list1, ListNode* list2){

    ListNode dummy(-1);
    ListNode* tail = &dummy;

    while(list1 != NULL && list2 != NULL){

        if(list1->val <= list2->val){

            tail->next = list1;
            list1 = list1->next;
        }
        else{

            tail->next = list2;
            list2 = list2->next;
        }

        tail = tail->next;
    }

    if(list1 != NULL)
        tail->next = list1;
    else
        tail->next = list2;

    return dummy.next;
}

Time Complexity: O(n + m)
Space Complexity: O(1)
