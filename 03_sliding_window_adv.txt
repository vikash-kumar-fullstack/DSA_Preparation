1. LONGEST REPEATING CHARACTER REPLACEMENT

---

Problem:
Given a string s and integer k.

You can replace at most k characters in the string so that
all characters in the substring become the same.

Return the length of the longest possible substring.

Example:
s = "AABABBA"
k = 1

Output:
4

Explanation:
"AABA" can become "AAAA" with 1 replacement.

---

## KEY IDEA

Maintain a sliding window where:

window_size - most_frequent_character <= k

Meaning:

characters_to_replace = window_size - maxFrequency

If replacements ≤ k → window is valid.

---

## SLIDING WINDOW STRATEGY

Maintain:

left pointer
right pointer
frequency array
maxFrequency

If window becomes invalid:

(window_size - maxFrequency > k)

Shrink window from left.

---

## C++ CODE

int characterReplacement(string s, int k) {

```
vector<int> freq(26,0);

int left = 0;
int maxFreq = 0;
int ans = 0;

for(int right=0; right<s.length(); right++){

    freq[s[right]-'A']++;

    maxFreq = max(maxFreq, freq[s[right]-'A']);

    while((right-left+1) - maxFreq > k){

        freq[s[left]-'A']--;
        left++;
    }

    ans = max(ans, right-left+1);
}

return ans;
```

}

Time Complexity: O(n)
Space Complexity: O(1)

2. MINIMUM SIZE SUBARRAY SUM

---

Problem:
Find the minimum length subarray whose sum ≥ target.

Example:

target = 7
nums = [2,3,1,2,4,3]

Output:
2

Explanation:
Subarray [4,3] = 7

---

## KEY OBSERVATION

Array contains positive numbers.

If sum ≥ target:
Shrink the window to minimize length.

---

## VARIABLE SLIDING WINDOW STRATEGY

Expand window by moving right pointer.

Add element to sum.

If sum ≥ target:
Update answer.
Shrink window from left.

---

## C++ CODE

int minSubArrayLen(int target, vector<int>& nums) {

```
int left = 0;
int sum = 0;
int ans = INT_MAX;

for(int right=0; right<nums.size(); right++){

    sum += nums[right];

    while(sum >= target){

        ans = min(ans, right-left+1);

        sum -= nums[left];
        left++;
    }
}

if(ans == INT_MAX)
    return 0;

return ans;
```

}

Time Complexity: O(n)
Space Complexity: O(1)
