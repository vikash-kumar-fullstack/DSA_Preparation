1. STACK PATTERN

---

Stack follows LIFO (Last In First Out).

Operations:

push()
pop()
top()

Used when:

* Next greater/smaller element
* Balanced parentheses
* Monotonic behavior
* Processing elements from right

---

2. VALID PARENTHESES

---

Problem:
Check if parentheses string is valid.

Rules:

* Every opening bracket has a matching closing bracket
* Order must be correct

---

## APPROACH

1. Push opening brackets into stack

2. For closing brackets:

   * Check if stack is empty → invalid
   * Check if top matches → pop
   * Otherwise → invalid

3. At end:

   * If stack empty → valid

---

## C++ CODE

bool isValid(string s){

```
stack<char> st;

for(char ch : s){

    if(ch=='(' || ch=='{' || ch=='[')
        st.push(ch);

    else{

        if(st.empty())
            return false;

        if((ch==')' && st.top()!='(') ||
           (ch=='}' && st.top()!='{') ||
           (ch==']' && st.top()!='['))
            return false;

        st.pop();
    }
}

return st.empty();
```

}

Time Complexity: O(n)
Space Complexity: O(n)

3. NEXT GREATER ELEMENT

---

Problem:
Find next greater element on right side.

---

## APPROACH (MONOTONIC STACK)

Traverse from right → left.

Steps:

1. Remove smaller elements from stack
2. Top gives next greater
3. If empty → -1
4. Push current element

---

## C++ CODE

vector<int> nextGreaterElement(vector<int>& nums){

```
int n = nums.size();
vector<int> ans(n);
stack<int> st;

for(int i=n-1;i>=0;i--){

    while(!st.empty() && st.top() <= nums[i])
        st.pop();

    if(st.empty())
        ans[i] = -1;
    else
        ans[i] = st.top();

    st.push(nums[i]);
}

return ans;
```

}

Time Complexity: O(n)
Space Complexity: O(n)
